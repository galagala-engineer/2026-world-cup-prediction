# 鹅鸭杀2026美加墨世界杯胜负预测下注规划系统

## 📜 规则项说明

| 规则项 | 说明 |
|--------|------|
| 每日获得 | 2000应援券 |
| 胜利收益 | 3倍下注金额的竞猜币 |
| 失败收益 | 获得1倍数量的竞猜币 |
| 最低下注 | 10应援券 |
| 下注倍数 | 必须为10的倍数 |

### 🎯 目标优先级

1. **获得竞猜币最大化**
2. **世界杯成功预测次数达到30场及以上**
3. **连续成功预测胜负结果10场及以上**

### ⚖️ 边界规则

1. 每次下注的应援券数量占持有应援券数量的占比在**60%-80%**，具体保留多少占比的应援券数量要根据前一天模型预测结果的准确度进行调整
2. 预测的置信度**大于等于60%**则按照置信度分配下注的应援券数量，胜负预测置信度**低于60%**的一律只下注10应援券

基于AI的2026年美加墨世界杯胜负预测与下注规划工具，专为鹅鸭杀玩家打造。

## 🎮 功能特点

- **胜负预测**：AI分析球队数据，预测每场比赛胜负平结果
- **置信度显示**：直观展示预测结果的置信程度
- **智能下注规划**：根据置信度动态分配应援券（60%-80%）
- **凯利公式优化**：科学计算每场比赛最佳下注金额
- **中英双语**：支持中文/English切换
- **响应式设计**：支持电脑、平板、手机访问

## 📱 使用方法

1. 打开 `index.html` 或 `index_api.html`
2. 输入应援券数量（最少10）
3. 点击"生成预测结果"
4. 查看预测结果、置信度和下注建议
5. 可提交反馈意见
