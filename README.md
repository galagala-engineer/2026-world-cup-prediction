# 鹅鸭杀2026世界杯预测系统

基于AI的2026年美加墨世界杯胜负预测与下注规划工具，专为鹅鸭杀玩家打造。

## 🎮 功能特点

- **胜负预测**：AI分析球队数据，预测每场比赛胜负平结果
- **置信度显示**：直观展示预测结果的置信程度
- **智能下注规划**：根据置信度动态分配应援券（60%-80%）
- **凯利公式优化**：科学计算每场比赛最佳下注金额
- **中英双语**：支持中文/English切换
- **响应式设计**：支持电脑、平板、手机访问

## 📱 使用方法

1. 打开 `index.html` 或 `index_api.html`
2. 输入应援券数量（最少10）
3. 点击"生成预测结果"
4. 查看预测结果、置信度和下注建议
5. 可提交反馈意见

## 📂 文件说明

- `index.html` - 主页面
- `index_api.html` - 备用页面

## 🎯 使用场景

- 世界杯期间实时预测比赛结果
- 鹅鸭杀应援券下注策略参考
- 足球赛事分析学习

## ⚠️ 注意事项

- 仅供娱乐参考，不构成投资建议
- 系统准确率46.5%（基于129场比赛验证）
