# 鹅鸭杀2026美加墨世界杯胜负预测下注规划系统

## 📜 报告摘要

本项目针对2026美加墨世界杯足球比赛胜负预测问题，构建了基于宏观经济视角与微观球员视角的双维度评估模型。项目成功实现了从Python后端到前端网页的完整部署，为鹅鸭杀玩家提供实时预测与下注规划服务。

**核心成果：**
- 构建了约阿希姆·克莱门特宏观经济方法论与迈克尔·布鲁诺微观球员方法论相结合的评估体系
- 开发了Python预测模型（MLP+XGBoost集成）及JavaScript前端预测引擎
- 实现了凯利公式下注策略，采用四分之一凯利系数（0.25）降低风险
- 动态下注比例根据置信度调整（60%-80%），确保资金合理分配
- 新增战意因子系统，优化爆冷事件预测
- 系统准确率46.5%（基于129场比赛验证）
- 成功部署至GitHub Pages，支持中英双语，适配多端访问

## 1 规则项说明

### 1.1 基础规则

| 规则项 | 说明 |
|--------|------|
| 每日获得 | 2000应援券 |
| 胜利收益 | 3倍下注金额的竞猜币 |
| 失败收益 | 获得1倍数量的竞猜币（返还本金） |
| 最低下注 | 10应援券 |
| 下注倍数 | 必须为10的倍数 |

### 1.2 目标优先级

1. **获得竞猜币最大化**
2. **世界杯成功预测次数达到30场及以上**
3. **连续成功预测胜负结果10场及以上**

### 1.3 边界规则

1. 每次下注的应援券数量占持有应援券数量的占比在**60%-80%**，具体保留多少占比的应援券数量要根据前一天模型预测结果的准确度进行调整
2. 预测的置信度**大于等于60%**则按照置信度分配下注的应援券数量，胜负预测置信度**低于60%**的一律只下注10应援券

## 2 问题定义

### 2.1 业务背景

参与鹅鸭杀2026世界杯球迷应援活动，目标是最大化竞猜币收益以兑换奖励。系统需要为用户提供科学的比赛预测和下注建议。

### 2.2 预测对象

2026年6月14日-6月28日共15天比赛，涵盖多场世界杯小组赛及热身赛，共计68场比赛。

## 3 分析思路

### 3.1 专家方法论融合

**约阿希姆·克莱门特宏观经济视角（权重35%）：**
- FIFA排名（65%）：近期竞技实力，最核心指标
- 东道主优势（15%）：主场效应
- 气候适应性（10%）：训练条件影响
- 人均GDP（5%）：衡量足球基础设施投入能力
- 人口规模（5%）：潜在人才池大小

**迈克尔·布鲁诺微观球员视角（权重65%）：**
- 明星球员评分（45%）：巨星表现决定上限，最核心指标
- 核心球员状态（20%）：基于身价的球员状态评估
- 历史战绩（20%）：近5年大赛表现
- 教练战术适配（8%）：战术体系与球员匹配度
- 阵容年龄结构（7%）：黄金年龄25-30岁球员占比

### 3.2 问题分解模型

```
比赛结果 = f(宏观经济因素 × 权重 + 微观球员因素 × 权重 + 随机因素)
```

### 3.3 关键假设

1. 球队综合评分可预测比赛结果（0-100分制）
2. 置信度越高，下注比例越大（60%-80%动态调整）
3. 运气因素约占比赛结果的45%-50%
4. 下注金额必须为10的倍数，最低10应援券
